# Employee Management System

## Description

A Java console-based Employee Management System that allows users to manage employee records using CRUD (Create, Read, Update, Delete) operations. The application is developed using Object-Oriented Programming (OOP) principles and stores employee data using ArrayList.

## Features

- Add Employee Records
- View Employee Details
- Update Employee Information
- Delete Employee Records
- Search Employee by ID
- Menu-Driven Interface
- ArrayList-Based Data Storage

## Technologies Used

- Java
- Object-Oriented Programming (OOP)
- ArrayList
- Scanner Class
- Loops
- Conditional Statements

## How to Run

Compile:

```bash
javac Employee.java EmployeeManagementSystem.java
```

Run:

```bash
java EmployeeManagementSystem
```
